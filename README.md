# 🚀 LinkedIn Profile Booster AI Auditor

An AI-powered local Chrome Extension that automatically scans your LinkedIn profile, runs a comprehensive data audit, and completely rewrites your Headline and "About" sections using optimized, high-converting industry keywords.

---

## 🛠️ How to Install (60-Second Manual Setup)

Since this is an exclusive developer-built optimization utility, it is loaded directly into your browser without going through the public Chrome Web Store. Follow these simple steps:

### Step 1: Download and Unpack the Tool
1. Click the green **Code** button at the top of this GitHub repository page.
2. Click **Download ZIP**.
3. Locate the downloaded file on your computer and **Extract/Unzip** it into a permanent folder (e.g., your Desktop folder).

### Step 2: Turn on Developer Mode in Chrome
1. Open Google Chrome.
2. Go to the URL bar, type `chrome://extensions` and hit **Enter**.
3. In the top-right corner, toggle **Developer mode** to **ON**.

### Step 3: Load the Extension
1. Click the **Load unpacked** button in the top-left corner.
2. Select the extracted folder containing the extension files (the folder that has `manifest.json` inside it).
3. Click **Open** or **Select**. The extension icon will now appear in your browser's toolbar tray!

## 📈 How to Run Your First Optimization Audit

1. Navigate to your personal **LinkedIn Profile page**.
2. Click the puzzle piece extension icon in your Chrome toolbar tray and select **Profile Booster AI Auditor**.
3. Click the blue **Scan Current Profile** button. 
4. Scan the secure **UPI QR Code** using any banking app (GPay, PhonePe, Paytm, or BHIM) to unlock your premium AI copywriting report.
5. Input your transaction reference/UTR number and hit **Submit Verification** to instantly generate your highly optimized LinkedIn copy!
